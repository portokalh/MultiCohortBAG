# Logs and Manifests

This folder contains selected small provenance files for publication audit.

Included files focus on Figure 10 / transferability manifests, QC summaries, and compact metrics tables. Large prediction tables, raw model outputs, checkpoints, raw data, and exploratory logs are intentionally not included.

Paths beginning with `$WORK` refer to the local project work root used during analysis.

The selected files document:
- cross-cohort transferability output generation,
- feature-set-specific transfer grids,
- compatibility/QC summaries,
- raw predicted-age versus cBAG transfer metrics.
